# Liver Mini Pipeline

This repository contains a beginner-friendly, reproducible bioinformatics pipeline
for liver disease biomarker discovery using RNA-seq, clinical features, AI models,
and explainability.

## Structure
- notebooks/: exploratory analysis and learning notebooks
- pipeline/: Nextflow pipeline for reproducible execution
- docker/: Docker container for consistent environments
- data/: input data (not tracked)
- results/: outputs (not tracked)
